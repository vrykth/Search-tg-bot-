# Search TG Bot

> Прототип поискового бота для Telegram. Ищет музыку, аниме, фильмы и всё остальное через AI.  
> Сделан на коленке в Termux, но работает 🔥

---

## Что умеет

- 🎵 **Музыка** — Яндекс.Музыка, VK Music, Last.fm
- 🎌 **Аниме** — Shikimori (рейтинги, серии, описания)
- 🎬 **Фильмы** — Кинопоиск
- 🔍 **Всё остальное** — Яндекс + DuckDuckGo + AI-анализ
- 🤖 **AI** — анализирует запрос и пишет красивый ответ через OpenRouter

---

## Стек

Python + `pyTelegramBotAPI` + `requests` + `BeautifulSoup`.  
Без тяжёлых библиотек — работает даже в Termux на Android.

---

## Быстрый старт

### 1. Установка (Termux / Linux / Windows)

```bash
pip install pyTelegramBotAPI requests beautifulsoup4
```

### 2. Настройка

Открой `bot.py` и впиши свои данные в самом начале:

```python
BOT_TOKEN           = "твой_токен_от_@BotFather"
ADMIN_ID            = 1234567890          # твой Telegram ID
OPENROUTER_API_KEY  = "sk-or-v1-..."      # ключ с openrouter.ai
```

Остальное можно оставить по умолчанию.

**Где взять:**
- Токен — пиши [@BotFather](https://t.me/BotFather)
- Свой ID — [@userinfobot](https://t.me/userinfobot)
- OpenRouter ключ — [openrouter.ai/keys](https://openrouter.ai/keys) (есть бесплатные модели)

### 3. Запуск

```bash
python bot.py
```

Готово. Бот онлайн.

## Как пользоваться

Просто пиши боту любой запрос — он сам поймёт, что искать:

| Что искать | Пример запроса |
|------------|----------------|
| Музыка | `трек Монеточка` |
| Аниме | `аниме про школу` |
| Фильм | `фильм Дюна` |
| Всё остальное | `как приготовить борщ` |

Команды:
- `/start` — приветствие
- `/help` — справка
- `/search <запрос>` — явный поиск
- `/admin` — панель админа (только для тебя)

---

## Админка

Команда `/admin` открывает меню:
- 📢 Рассылка всем юзерам
- 📊 Статистика (кто сколько запросов сделал)
- 🧠 Смена AI-промпта
- 🔌 Включение/выключение провайдеров
- 📋 Логи запросов
